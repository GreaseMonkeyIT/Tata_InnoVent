How to connect (run on the box)

This page reads live metrics straight from Prometheus — no changes to the running system. Pick either path, then set the field above to match.

A · port-forward (simplest; relies on Prometheus' default CORS):

kubectl port-forward -n observability svc/prom-kube-prometheus-stack-prometheus 9090:9090
# then open this file and leave the base URL as  http://localhost:9090

B · kubectl proxy (zero-CORS; serves this page and Prometheus from one origin):

kubectl proxy --www=tools --www-prefix=/ui/ --port=8001
# browse to  http://localhost:8001/ui/pod-resources.html
# and set the base URL to (relative, no host):
#   /api/v1/namespaces/observability/services/prom-kube-prometheus-stack-prometheus:9090/proxy

If the dot turns red with a network/CORS error, use path B (same-origin) — it sidesteps CORS entirely.
